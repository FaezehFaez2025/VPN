### Simple Python VPN (toy)

**Warning:** no encryption/authentication. Use only for testing.

### Server (Linux VPS)

```bash
sudo python3 vpn.py server
```

### Laptop (client)

- **macOS:**

```bash
sudo python3 vpn.py client <SERVER_PUBLIC_IP>
```

- **Linux:**

```bash
sudo python3 vpn.py client <SERVER_PUBLIC_IP>
```
